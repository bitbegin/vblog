# 数据解析

原始数据

6a146f6d6e6900000000000000020000000000000001

解析：

6a          ; OP_RETURN 
14          ; length
6f6d6e69    ; string "omni"
0000        ; version
0000        ; Simple Send 
00000002  ; omni test
0000000000000001  ; value
